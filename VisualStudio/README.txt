Note:

If you are importing Prabir_DarkTheme_vs2010_with_Resharper_support.vssettings.
Two key bindings are changed.
 Ctrl + U -> remove and sort using, instead of converting text to lower case
 Ctrl + , -> Will use the default visual studio navigate to instead of Resharpers go to recent files.

If you don't want the keybindings to be changed please untick keyboard when importing settings.

Prabir's Dark Theme for Notepad++ requires Consolas Font which comes preinstalled with Windows Vista and up. But incase you want to use it for your Windows XP, Windows Server 2003 or others you can either intall this one or download from https://www.ohloh.net/p/themes/download?filename=ConsolasFontPack.zip or from the official Microsoft website http://www.microsoft.com/downloads/details.aspx?familyid=22e69ae4-7e40-4807-8a86-b3d36fab68d3

Themes for others Editors such as eclipse, notepad++ and vim can be downloaded at https://www.ohloh.net/p/themes/download
