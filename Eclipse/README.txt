Changes:
	Added better support for XML theme

prabir_darktheme : Dark Eclipse theme with default font
prabir_darktheme_consolas: Dark Eclipse theme with Consolas Font

Recommended Theme: prabir_darkthem_consolas is the recommended theme as Consolas font is better suited for code viewing.

You can find the source code repository at http://github.com/prabirshrestha/dark-themes

Note:
prabir_darkthem_consolas requires Consolas Font which comes preinstalled with Windows Vista and up. But incase you want to use it for your Windows XP, Windows Server 2003 or others you can either intall this one or download from https://www.ohloh.net/p/themes/download?filename=ConsolasFontPack.zip or from the official Microsoft website http://www.microsoft.com/downloads/details.aspx?familyid=22e69ae4-7e40-4807-8a86-b3d36fab68d3
